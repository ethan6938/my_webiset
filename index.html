JS Super-Calculator

The JavaScript Super-Calculator is a versatile and interactive web-based application that performs a wide range of mathematical operations beyond basic arithmetic. Designed with both functionality and user experience in mind, this tool showcases the flexibility of JavaScript in handling real-time calculations and dynamic UI updates.Built using a combination of HTML for structure, CSS for styling, and JavaScript for logic and interactivity, the Super-Calculator demonstrates how core web technologies can be used to create powerful and responsive tools. JavaScript drives all real-time computations, manages user input through event listeners, and dynamically updates the interface via DOM manipulation. Optional enhancements such as Bootstrap for design or localStorage for saving preferences can further expand its capabilities.

Lunch Game!

Lunch Game! is a light-hearted, interactive web- or Python-based game that blends fun with fast-paced decision-making. Designed to simulate the chaos of choosing or preparing lunch under pressure, it engages players through quick reflexes, colorful visuals, and playful mechanics.If built as a web game, it uses HTML for structure, CSS for visual styling, and JavaScript (or a game library like Phaser.js) to power game logic, player input, animations, and real-time updates. For a Python-based version, technologies like Pygame handle graphics rendering, input detection, and game loop mechanics. In either case, the game may involve dodging unhealthy food, collecting ingredients, or assembling orders within a time limit—making it both entertaining and surprisingly strategic.Optional enhancements may include sound effects, scoreboards using localStorage or JSON files, and responsive design for mobile play.

alien_invasion

Alien Invasion is a classic 2D arcade-style game developed using Python and the Pygame library, where the player controls a spaceship that must defend Earth from waves of descending aliens. This project serves as a foundational experience for learning game development fundamentals in Python and is widely recognized for introducing key programming concepts in an engaging way.Built entirely with Python, the game uses Pygame to manage the game loop, render sprites and backgrounds, detect collisions, handle keyboard input, and control enemy behavior. It showcases essential principles of object-oriented programming (OOP), including class-based design for ships, bullets, and aliens, as well as structured event handling and state management.Optional features such as score tracking, increasing difficulty, explosion effects, and sound integration can enhance gameplay and extend learning.

Galactic User Fetcher

Drag Race Game

The Drag Race Game is a high-speed, visually thrilling browser-based racing game that delivers an adrenaline-pumping experience. Built using HTML for structure, CSS for styling, and JavaScript with the Phaser.js game framework, the game combines responsive controls, smooth animations, and immersive visuals to engage players in fast-paced drag racing action.Players can customize their cars, choose from vibrant track themes (easy, medium, hard), and compete against AI opponents through multiple laps. JavaScript and Phaser handle everything from physics-based movement, collision detection, and lap tracking, to managing power-ups, boss battles, and difficulty scaling. The dynamic environments, animated UI elements, and real-time feedback create a polished and professional game experience.Optional enhancements include sound effects, leaderboards using localStorage or online storage, and animated trophies for winners—making this project a strong showcase of browser game development with interactive gameplay and rich design.
